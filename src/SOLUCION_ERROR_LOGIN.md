# ❌ Solución: "Email o contraseña incorrectos"

## 🔍 Por Qué Pasa Este Error

Estás intentando **INICIAR SESIÓN** pero no tienes ninguna cuenta creada todavía (es un proyecto nuevo de Supabase).

---

## ✅ Solución en 3 Pasos

### PASO 1: ¿Ejecutaste el Script SQL? ⚠️

**IMPORTANTE**: Si no ejecutaste el script SQL, hazlo AHORA:

1. Abre: **https://rqwyqipixtjnuubnnsmv.supabase.co**
2. Menú lateral → **SQL Editor**
3. Botón **"New query"**
4. Copia TODO de `/SETUP_SUPABASE_DESDE_CERO.sql`
5. Pega y presiona **"Run"**
6. Espera ver: `🎉 ¡CONFIGURACIÓN COMPLETA!`

Si **YA** lo ejecutaste, continúa al Paso 2 ⬇️

---

### PASO 2: Recargar la App

1. Presiona **F5** en tu navegador
2. Espera a que cargue completamente

---

### PASO 3: Usar "REGISTRARSE" (No "Iniciar Sesión")

En la pantalla que ves:

#### ❌ NO HAGAS ESTO:
```
Email: ___________
Contraseña: ___________
[Iniciar Sesión] ← NO PRESIONES ESTE
```

#### ✅ HAZ ESTO:
1. Busca un botón o link que diga **"REGISTRARSE"** o **"Crear cuenta"**
2. Debería estar cerca del botón "Iniciar Sesión"
3. **Haz clic en "REGISTRARSE"**

#### Luego completa:
```
Usuario: Ian (o el nombre que quieras)
Email: ianlihuel97@gmail.com (para ser developer)
Contraseña: la que tú quieras
[Registrarse] ← PRESIONA ESTE
```

---

### PASO 4: Ahora SÍ Inicia Sesión

Después de registrarte:
1. Usa el botón **"Iniciar Sesión"**
2. Email: `ianlihuel97@gmail.com`
3. Contraseña: la que acabas de crear
4. ¡Deberías entrar! ✅

## 🔍 Si Aún No Ves el Botón "REGISTRARSE"

Es posible que la interfaz haya cambiado. Busca:
- **"Crear cuenta"**
- **"Sign up"**
- **"Nuevo usuario"**
- Un **toggle/switch** que cambie entre "Login" y "Register"
- Un **link** debajo del botón de login

---

## 🆘 Si Sigue Sin Funcionar

Puede ser que las variables de entorno no se actualizaron. Intenta:

### Opción A: Limpiar Caché del Navegador
1. Presiona **Ctrl + Shift + R** (Windows/Linux)
2. O **Cmd + Shift + R** (Mac)
3. Esto recarga sin usar caché

### Opción B: Abrir en Ventana Incógnito
1. **Ctrl + Shift + N** (Chrome/Edge)
2. **Ctrl + Shift + P** (Firefox)
3. Abre la app de Beyblade
4. Intenta registrarte
