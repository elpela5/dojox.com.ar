## 🆘 Si Algo Sale Mal

### Error: "Invalid API key" o "Failed to fetch"
**Causa**: Las variables de entorno no se actualizaron.

**Solución**:
1. Verifica que ejecutaste el script SQL completamente
2. Recarga la app (F5) nuevamente
3. Si persiste, avísame y lo reviso

### Error: "Email already registered"
**Causa**: Ya intentaste registrarte antes.

**Solución**:
1. Haz clic en "Iniciar Sesión" en lugar de "Registrarse"
2. Usa el email y contraseña que usaste antes

### Error: "relation does not exist"
**Causa**: No ejecutaste el script SQL o falló.

**Solución**:
1. Ve a Supabase → SQL Editor
2. Ejecuta `/SETUP_SUPABASE_DESDE_CERO.sql` nuevamente
3. Verifica que veas los mensajes de éxito

### Los torneos se borran al recargar
**Causa**: Las políticas RLS no se crearon.

**Solución**:
1. Ejecuta el script SQL nuevamente
2. Verifica que veas "✅ Políticas RLS creadas"
3. Recarga la app (F5)
