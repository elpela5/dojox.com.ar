
  # Beyblade Tournament Tracker App

  This is a code bundle for Beyblade Tournament Tracker App. The original project is available at https://www.figma.com/design/40SVTUJfbixX0VKVqkxt61/Beyblade-Tournament-Tracker-App.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  
  Versión final y corregida. Incorpora diversas secciones y funciones
●meta actual local: con auto-carga de la informacion de los decks 10 mejores jugadores de cada torneo
●torneos: Los administradores pueden crear torneos ya sea en formato suizo, eliminación directa o combinado, en el cual se hará una primera parte suiza, top cut de los primeros lugares, y finalmente eliminación directa entre dichos usuarios. Cuando se crea el torneo se puede elegir a que liga estarán asociados los resultados.
●coleccion: los usuarios pueden llevar recuento de las piezas que tienen y armar decks en base a dichas piezas.
●clasificacion: clasificacion general con tiers. creación de ligas y sus respectivas tablas de clasificacion independientes.
●comunidad: informacion de la comunidad y flyer sobre próximos torneos.
●perfil: ver mi perfil, cambiar nombre de usuario o contraseña.
●administracion: ver a los usuarios registrados, poder cambiar sus claves o nombres de usuario en caso de necesidad. Modificar los permisos de usuarios: administrador, juez, jugador.
